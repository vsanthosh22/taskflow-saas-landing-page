# TaskFlow – SaaS Project Management Landing Page

TaskFlow is a modern, conversion-focused SaaS landing page designed for a project management platform. It highlights product features, pricing, and onboarding flow with a clean and structured UI.

## 🚀 Live Demo



## 📌 Features

- 🎯 Conversion-focused hero section
- 📊 Feature showcase with grid layout
- ⚙️ "How It Works" onboarding flow
- 💰 Transparent pricing section
- ⭐ Testimonials for social proof
- 📢 Strong call-to-action sections
- 📱 Fully responsive design

## 🧩 Sections Included

1. Hero Section
2. Trusted By (Social Proof)
3. Features Grid
4. How It Works
5. Pricing Plans
6. Testimonials
7. Call-to-Action Banner
8. Footer
